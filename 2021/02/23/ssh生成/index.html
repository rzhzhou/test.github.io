如何生成SSH key

SSH key提供了一种与GitHub通信的方式，通过这种方式，能够在不输入密码的情况下，将GitHub作为自己的remote端服务器，进行版本控制.

步骤

  • 检查SSH keys是否存在
  • 生成新的ssh key
  • 将ssh key添加到GitHub中

第一步：检查ssh keys是否存在

进入到ssh目录下(在根目录下执行)

输入ls查看该目录下文件，如果有文件 id_rsa.pub 或 id_dsa.pub (例如下)，则直接进入步骤3将SSH key添加到GitHub中，否则进入第二步生成SSH key

第二步: 生成新的ssh key
